## 소개
<br/>
<br/>



기본적인 CRUD를 이용한 게시판 프로젝트

<br/>

버튼을 이용한 게시판 기본 기능 구현

<br/>
<br/>
<br/>
<br/>




#### 구현 메소드
<br/>

* insert : 게시글 작성
* selectAll : 전체 게시글 목록 조회
* selectOne : 특정 게시글 상세 조회
* update : 게시글 수정
* delete : 게시글 삭제
* updateCnt : 게시글 조회수 카운트
